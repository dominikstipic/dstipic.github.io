Review

Study only has 47 subjects which are sampled from only one geographic region. They use 3 control variables: sex, age and education level. In order to prevent Simpson paradox or the unusual effect which hidden variables have on the relationship between the target variable and the independent variable they need to create a groups based on this 3 control variables. I don't know, sex would be binary, age would be grouped in 3-4 subgroups and education level would be primary school, high-school, bachelor, master, no school.... The best option would be to create a randomized trails where the enumeration of possible confounding variables wouldn't be necessary. Good demonstration of the Simpson paradox could be found here gif.
Basically, when the regression is fitted on the overall data we have negative relationship between variables. When we introduce a new cofounding variable which breaks the data into 5 groups we get opposite relationship between independent and dependent variable. This would mean to allocate the coffee treatment randomly to units. Population would be so large that in contains all kinds of genetic and cultural variabilities.
